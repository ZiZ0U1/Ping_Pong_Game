# Ping_Pong_Game

A Simple C++ Game.

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/ZiZ0U1/Ping_Pong_Game.git
   cd Ping_Pong_Game
   ```

2. Build the project
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

3. Run the game
   ```bash
   ./ping_pong
   ```
